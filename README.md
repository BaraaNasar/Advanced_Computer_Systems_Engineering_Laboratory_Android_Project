# How to Run the Finance App

**تشغيل التطبيق في Android Studio:**

1. افتح برنامج **Android Studio**.
2. من القائمة: **File** -> **Open**.
3. اختر هذا المجلد:
   `C:\Users\USER\Desktop\Advanced_Computer_Systems_Engineering_Laboratory_Android_Project`
4. انتظر حتى ينتهي التحميل بالكامل.
5. اضغط على زر **تشغيل (Run)** الأخضر في الأعلى.

المشروع جاهز للتشغيل مباشرة.
