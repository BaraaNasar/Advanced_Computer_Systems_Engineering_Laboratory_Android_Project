# How to Run the Finance App

**Running the App in Android Studio:**

1. Open **Android Studio**.
2. From the menu: **File** -> **Open**.
3. Select this folder:
   `C:\Users\USER\Desktop\Advanced_Computer_Systems_Engineering_Laboratory_Android_Project`
4. Wait until the project finishes loading completely.
5. Click the green **Run** button at the top.

The project is ready to run immediately.
